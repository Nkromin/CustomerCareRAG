IT Security & Access Policy
1. Password Policy

Minimum 12 characters.

Must include uppercase, lowercase, number, special character.

Change every 90 days.

2. Multi-Factor Authentication

Mandatory for:

VPN access

Email login

Finance systems

3. Access Control

Access is granted based on role-based access control (RBAC).

4. Incident Reporting

Employees must report security incidents within 24 hours.

5. Prohibited Actions

Installing unauthorized software

Sharing credentials

Downloading pirated content
